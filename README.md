# Campus AI 🧠📉

An intelligent, AI-powered system designed for educational institutions to read, analyze, and monitor students' textual inputs, identifying their psychological state and stress levels to provide proactive mental health support.

## 🚀 About The Project

#CampusAI helps universities monitor student well-being by analyzing their text inputs, posts, or feedback across campus platforms. The system processes natural language to detect underlying signs of sadness, anxiety, burnout, or depression while maintaining student anonymity.

## ✨ Key Features

* **Sentiment Analysis:** Classifies student text into positive, negative, or neutral categories.
* **Psychological Indicator Detection:** Flags specific linguistic markers for stress, burnout, and depression.
* **Early Warning System:** Generates instant alerts for campus counselors when critical emotional distress is detected.
* **Interactive Dashboard:** Visualizes overall campus mental health trends for administrators.
* **Privacy-First Design:** Anonymizes and encrypts student data to ensure strict confidentiality.

## 🛠 Tech Stack

* **AI & NLP:** Python, Hugging Face Transformers, Scikit-learn
* **Backend API:** FastAPI
* **Frontend:** React.js

## ⚙️ Quick Start

### 1. Clone the repository
```bash
git clone github.com
cd campus-ai
```

### 2. Install dependencies
```bash
pip install fastapi uvicorn transformers torch
```

### 3. Run the server
```bash
python main.py
```
The API will be live at `http://127.0.0.1:8000`. You can access the interactive API docs at `127.0.0`.
