# Dataset Card — CampusCare Large Dataset

## Dataset name

`campuscare_large_dataset.csv`

## Size

60,000 text messages.

## Purpose

The dataset supports a deep learning course project about routing anonymous university support messages into human-support queues.

## Source and creation method

This is a custom synthetic dataset generated by `src/generate_big_dataset.py`. The generator uses realistic campus support-message patterns, university-course terms, deadline/exam scenarios, channel metadata, and controlled text augmentation. The goal is to create a large reproducible dataset for model comparison when internet access or external data approval is not available.

## Classes

| Label | Count | Description |
|---|---:|---|
| `normal` | 24,000 | Routine/positive/admin messages |
| `academic_stress` | 18,000 | Study pressure, deadline anxiety, grade concerns |
| `emotional_distress` | 12,000 | Sadness, isolation, hopelessness, persistent distress |
| `urgent_human_review` | 6,000 | Safety-risk language that must be escalated |

## Additional challenge set

`campuscare_challenge_test.csv` contains 1,200 ambiguous examples designed to reveal model weaknesses. It includes messages where lexical cues can mislead the model, for example:

- normal messages that mention stress but are routine,
- academic messages that sound emotionally intense but are about grades,
- emotional distress messages that explicitly say they are not an immediate danger,
- urgent messages mixed with academic context.

## Columns

| Column | Meaning |
|---|---|
| `row_id` | numeric row index |
| `message_id` | synthetic message identifier |
| `text` | student message text |
| `label` | target class |
| `severity_score` | synthetic severity indicator from 0 to 1 |
| `channel` | message source channel |
| `student_year` | student academic year |
| `source` | dataset source tag |
| `is_synthetic` | whether the record is synthetic |

## Ethical limitations

This dataset is not clinical evidence and is not a substitute for professional mental-health assessment. The project must be framed as human-in-the-loop routing, not diagnosis or automated intervention. False negatives in the urgent class are especially risky, so any real deployment would require human review, conservative thresholds, institutional approval, and crisis-support protocols.

## Reproducibility

Regenerate the dataset with:

```bash
python src/generate_big_dataset.py --rows 60000
```
